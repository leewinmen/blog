    • 极速翱翔的3dsMax精简心得
    2022-07-21|实用经验
    字数总计: 2.1k|阅读时长: 6 分钟| 阅读量:

    不知不觉已经做了11个版本的精简工作，极速翱翔不会编程，通过研究分析删减无关文件，还有研究其注册表，实现了大幅度的精简。前后完成了3dsMax 2013—2023共11个版本的精简，非常欢迎大家使用3dmax极速翱翔精简版。

    本人非专业软件开发，纯用爱发电，且用且珍惜。

    遇到任何问题请先阅读以下文档，内容整理自各位网友通过邮件，公众号向我提问，可以解决98%的疑问和安装卸载问题。如果文档没提及，那这个问题也有有人曾向我提出，但至今没有好的解决方法。

    关于精简版

    1.解压密码

    大部分是blog.darler.cn
    但也需要擦亮眼睛，因为这个我绝不会藏着

    2.精简版精简了什么

    这是大家最关心的，精简版主要精简了运行库，可选安装插件，3DMAX内置预设，其他语言的相关文件，这些体积非常大，这也是精简版体积可以比原版减少90%的主要原因。

    3.为什么最老只做到2013版

    3DMAX 2013是目前可以使用VRAY 4.1且有汉化的最老版本。而对于32位的老版本，他们体积本身就不大，我也就懒得精简了。

    4.对各个版本的评价

    极速翱翔非室内设计相关行业，我的选择会安装2022版，只是用来转换模型。
    如果你的电脑性能一般，需要体积小一点的，可以使用2013版。
    我发布2个多月时间，数据显示2016和2019下载人数最多。
    但是2021发布后，很明显的感知是2021完胜2020。
    根据后台反馈，2020问题最多，2020是Autodesk更新破解方法和压缩方法的第一年，在安装过程如果没有一次成功，出现1625，1603。请直接放弃安装

    5.切换为中文

    极速翱翔制作了默认启动中文的补丁，但还是更建议从开始菜单使用简体中文的快捷方式。

    6.新版本一些变化

    2015及之前的3DMAX都分为3DSMAX和3DS MAX DESIGN2个安装包。
    从2016开始，3DMAX和MAX DESIGN都在一个安装包。
    安装后第一次启动会让你进行选择。

    7.2017注册方式的改变

    序列号和密钥在启动软件时才输入（2016及之前的版本是在安装软件时就要输入）

    8.关于汉化和精简VRay

    特别是3DMAX2020的汉化版VRay
    我不会！！！我只是做了个3DMAX，周边的软件原理啥的都不一样
    但是2021及配套汉化VRay4.32已经出来了，欢迎到公众号中下载。

    9.稳定性和功能完善问题

    要明白，精简版不是残废版，只删掉不要的部分
    装不上精简版的，要看看原版能不能装上，原版安装也有问题的，电脑系统本身问题

    10.3dsMax不集成VRay

    第一，我做精简版，做的是减法，你还让我增加，这个有违初衷。
    第二，每个人用的VRay版本不一致，总不可能让我一个版本的3DMAX，同时另外再做集成VRay3.4，集成VRay3.6，集成VRay4.1多个版本吧。如果你很想要，那你就得定制了。

    11.3dsMax不集成注册

    我只是做精简，要是集成了注册，就成盗版传播了。软件破解没任何难度，方法已写在上面，你自己就能完成。

    安装和卸载

    接下来说一些常见的安装问题

    1.常见运行库问题

    在没有完整 VC++，DX等运行库的机器上，会遇到缺少d3dx9_43.dll，vcredist，net 的问题，不能导致安装和启动不能进行。

    Win10可无视所有运行库问题，因为Win10系统会自动帮你解决缺少这些的问题，Win7/8部分系统也可以自己联网下载。这也是注明Win10（推荐）的原因
    为此专门制作了针对精简版的3DsMax2013–2020所需要的运行库

    1.缺少msvcp.dll,vcruntime


    运行VCRedist_x64/x86文件夹里面的进行安装，有提示具体版本的按照所缺少的版本就行。

    2.缺少d3dx9_.dll

    运行DirectX文件夹里面的DXSETUP.exe进行安装，3dsMax2016之前的才会提示需要，3dsMax2017之后不需要了。

    3.缺少.NET，net framework

    打开对应版本的NET Framework文件夹里面进行安装，大多数时候2.5没有windows会自行下载。

    运行库打包下载地址，如链接失效请在公众号联系我更新
    天翼网盘 访问码：l1w5
    百度网盘 提取码：jsax
    从极速翱翔的服务器下载

    运行库单独下载（缺少哪个，下载哪个）
    VCRedist
    DirectX
    NET Framework 3.5 ; NET Framework 4 ; NET Framework 4.5 ; NET Framework 4.7 ; NET Framework 4.8

    2.显示载入界面后无法启动

    这个问题一般出现在2016后的版本，解决方法：手动安装 \x64\CLIC\CLIC_x64_Release.msi

    3.运行Setup.exe弹出以下界面

    There is a problem with this Windows Installer package.A program run as part of the steup…..

    解决方法：运行\x64\max\3dsMax.msi这个msi文件来安装

    4.安装路径不要有中文


    补充一下：绿色版解压路径也不能有中文。

    5.注册时遇到问题

    you need to apply patch when licencescreen appears.

    运行注册机出现以上界面，需要先确保2点：
    1.已经断网，杀毒软件也提前关了。
    2.等激活界面出现时，再以管理员身份运行注册机。

    6.1603，1625错误


    2020-2023均可能出现1603，1625，这一个是系统问题，遇到建议放弃。
    如果遇到，你也可以尝试安装官方原版，
    当你安装官方原版仍然无法解决，你就会知道，真不是我精简导致的。

    关于对策，虽然有成功解决的案例，但无法做到百分百适合每台电脑。如果想要尝试解决，在开始之前先确保以下几点。
    1.讨论基于官方原版安装包
    2.运行库安装完整
    3.这台电脑是首次安装这个版本3dsMax，之前没有卸载过

    开始操作：
    1.使用Win10
    2.使用管理员系统账户(Administrator)
    3.运行使用x64\max\3dsMax.msi这个msi文件来安装

    7.系统管理员设置了系统策略

    系统管理员设置了系统策略，禁止进行此安装
    对此，请使用管理员系统账户进行操作

    8.3dsMax正确卸载

    （2020及之前的版本）
    答：使用Uninstall Tool。接下来我跟大家讲怎么找到这个工具：

    a.已安装3DMAX但未卸载
    开始菜单-Autodesk-Uninstall Tool

    b.在安装路径的（以2016为例）
    C:\Program Files (x86)\Common Files\Autodesk Shared\Uninstall Tool\R1

    c.已卸载或未安装3DMAX

    在安装包的（以2016为例）
    x64\max\Autodesk\3ds Max 2016\PFiles\CommFiles\Autodesk Shared\Uninstall Tool\R1

    9.错误126 Error 126

    错误代码126 - 找不到指定的模块。这个问题从2020开始直到2023仍然存在。

    无解！！！如果遇到，请尝试安装官方原版，
    当你安装官方原版仍然无法解决，你就会知道，真不是我精简导致的。

    10.NET运行异常 MAXScript

    MAXScript加载错误
    运行时错误：NET运行异常，未能加载/找不到指定模块

    无解！！！如果遇到，请尝试安装官方原版，
    当你安装官方原版仍然无法解决，你就会知道，真不是我精简导致的。
