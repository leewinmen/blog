我遇到的坑

1.GitHub创建博客报错404 There isn’t a GitHub Pages site here.

解决方案：

1.进入setting

2.找到GitHub Page,点击Change theme（设置之后即可访问）

2.Git发生Another git process seems to be running in this repository.

解决方案：

到你的Git文件夹去 ,然后会发现文件夹里面多了一个.git文件,打开.git文件，然后找到index.lock删除
